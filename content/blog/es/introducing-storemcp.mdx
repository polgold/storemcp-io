Si usaste Claude, ChatGPT o Cursor con MCP, ya conocés la magia: hacés una pregunta, la IA llama herramientas, las cosas pasan. Sin UI personalizada, sin dashboard, solo conversación.

Hasta hoy, esa magia se detenía en la puerta de WordPress.

## El vacío de WordPress

WordPress impulsa el ~43% de la web y una gran parte del comercio online. Pero conectar WordPress a la IA siempre implicó uno de estos tres compromisos:

1. **Construir una integración REST personalizada** — lenta, frágil, requiere mantenimiento cada vez que WooCommerce se actualiza.
2. **Pegar datos manualmente en el chat** — bien para un producto, un dolor para 1.000.
3. **Usar una función de IA incorporada en algún plugin** — generalmente solo "escribime una descripción de producto."

Ninguna es lo que los asistentes de IA deberían sentirse. Una IA que puede leer, escribir, analizar y actuar en toda tu tienda — eso es lo que queremos.

## Qué hace StoreMCP

StoreMCP es un único plugin de WordPress. Activalo, generá una API key, y pegá la URL en Claude, ChatGPT, Cursor o cualquier otro cliente compatible con MCP.

Obtenés al instante **más de 120 tools** que cubren:

- Productos, variaciones, categorías, tags, atributos
- Órdenes (con ciclo de vida completo — estado, notas, reembolsos)
- Clientes con historial de compras
- Cupones con todas sus restricciones
- Reportes de ventas y analytics
- Envíos, impuestos, pasarelas de pago
- Páginas, posts, media, menús, widgets
- Usuarios, reseñas, meta SEO
- Gestión de plugins y temas
- Webhooks, tools de sistema

Todo expuesto a través de la interfaz de herramientas de la IA. Preguntá "¿cuáles fueron mis 10 mejores productos el mes pasado?" — la IA llama la tool correcta, obtiene datos reales y te escribe la respuesta.

## ¿Por qué MCP?

Model Context Protocol es el estándar abierto al que convergieron todos los grandes asistentes de IA en 2025. No es propiedad de ningún proveedor. Construís un servidor MCP una vez y funciona con Claude.ai, Claude Desktop, Claude Code, ChatGPT Connectors, Cursor, VS Code, Windsurf y cualquier cliente futuro.

Eso importa para WordPress. No querés apostar tu flujo de trabajo a la API de un solo proveedor de IA.

## Open core

La versión free de StoreMCP es open source y está en wordpress.org. Te da:

- Gestión completa de contenido WordPress
- Navegación de productos (solo lectura)
- Reportes básicos
- Todas las tools de info del sitio

Si manejás un blog o sitio de contenido, el free es para siempre. Sin pantallas de molestia.

Pro y Agency agregan WooCommerce CRUD completo, analytics avanzado, operaciones masivas, multi-sitio y white-label. Precio anual — sin subscripción eterna, sin tiers ocultos.
