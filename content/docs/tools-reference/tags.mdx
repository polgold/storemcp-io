## store_mcp_list_tags

List product tags. **Tier:** Free.

### Parameters

| Parameter | Type    | Required | Default | Description |
|-----------|---------|----------|---------|-------------|
| search    | string  | no       | —       | Search by name |
| hide_empty| boolean | no       | false   | Hide unused tags |
| page      | integer | no       | 1       | Page number |
| per_page  | integer | no       | 50      | Items per page |

## store_mcp_get_tag

Get a tag by ID or slug. **Tier:** Free.

## store_mcp_create_tag

Create a tag. **Tier:** Pro. Parameters: `name`, `slug`, `description`.

## store_mcp_update_tag

Update a tag. **Tier:** Pro.

## store_mcp_delete_tag

Delete a tag. **Tier:** Pro.
