## store_mcp_list_posts

List posts. **Tier:** Free.

### Parameters

| Parameter | Type    | Required | Default | Description |
|-----------|---------|----------|---------|-------------|
| search    | string  | no       | —       | Title/content search |
| category  | integer | no       | —       | Category ID |
| tag       | integer | no       | —       | Tag ID |
| author    | integer | no       | —       | Author user ID |
| status    | string  | no       | publish | `publish`, `draft`, `pending`, `future` |
| after     | string  | no       | —       | ISO date |
| before    | string  | no       | —       | ISO date |
| page      | integer | no       | 1       | Page number |
| per_page  | integer | no       | 10      | Items per page |

Same as [Pages → create_page](/docs/tools-reference/pages#store_mcp_create_page) plus: `categories`, `tags`, `featured_media`, `format`.
