## store_mcp_list_widget_areas

List all registered widget areas / sidebars. **Tier:** Free.

## store_mcp_list_widgets

List widgets in a given area. **Tier:** Free.

## store_mcp_add_widget

Add a widget to an area. **Tier:** Free.

## store_mcp_update_widget

Update a widget's settings. **Tier:** Free.

## store_mcp_remove_widget

Remove a widget. **Tier:** Free.

<Callout type="info">
  On block themes, "widget areas" are block patterns and template parts —
  StoreMCP maps these transparently so the same tools work on both classic
  and block themes.
</Callout>
