## store_mcp_list_categories

List product categories. **Tier:** Free.

### Parameters

| Parameter | Type    | Required | Default | Description |
|-----------|---------|----------|---------|-------------|
| search    | string  | no       | —       | Search by name |
| parent    | integer | no       | —       | Filter by parent category ID |
| hide_empty| boolean | no       | false   | Hide categories with zero products |
| page      | integer | no       | 1       | Page number |
| per_page  | integer | no       | 50      | Items per page |

## store_mcp_get_category

Get a single category by ID or slug. **Tier:** Free.

## store_mcp_create_category

Create a category. **Tier:** Pro.

Parameters: `name`, `slug`, `parent`, `description`, `image`, `display`.

## store_mcp_update_category

Update a category. **Tier:** Pro.

## store_mcp_delete_category

Delete a category. **Tier:** Pro. Products are not deleted — only un-linked.
