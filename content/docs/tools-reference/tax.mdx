## store_mcp_create_tax_rate

Create a new tax rate. **Tier:** Pro.

### Parameters

| Parameter | Type    | Required | Description |
|-----------|---------|----------|-------------|
| country   | string  | yes      | ISO 2-letter country code |
| state     | string  | no       | State/province code |
| postcode  | string  | no       | Postcode range |
| city      | string  | no       | City name |
| rate      | string  | yes      | Tax rate (e.g. `21.0`) |
| name      | string  | yes      | Display name |
| class     | string  | no       | Tax class (defaults to standard) |
