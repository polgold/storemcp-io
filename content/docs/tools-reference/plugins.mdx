## store_mcp_deactivate_plugin

Deactivate a plugin. **Tier:** Pro.

## store_mcp_update_plugin

Update a plugin to the latest version. **Tier:** Pro.

## store_mcp_install_plugin

Install a plugin from the WordPress.org repo by slug. **Tier:** Pro.

<Callout type="warning" title="Install with care">
  Installing plugins via AI should be reserved for known, trusted slugs. A
  misheard plugin name could install the wrong plugin. Confirm before
  proceeding.
</Callout>
