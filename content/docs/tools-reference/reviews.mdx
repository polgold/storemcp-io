## store_mcp_list_reviews

List product reviews with filters. **Tier:** Free.

### Parameters

| Parameter | Type    | Required | Default | Description |
|-----------|---------|----------|---------|-------------|
| product   | integer | no       | —       | Product ID |
| status    | string  | no       | approved| `approved`, `hold`, `spam`, `trash`, `all` |
| rating    | integer | no       | —       | 1–5 |
| reviewer_email | string | no  | —       | Reviewer email |
| page      | integer | no       | 1       | Page number |
| per_page  | integer | no       | 10      | Items per page |
