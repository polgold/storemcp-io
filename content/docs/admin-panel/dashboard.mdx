The StoreMCP dashboard is the first screen you see in **WP Admin → StoreMCP**.

It shows:

- **Connection status** — whether any MCP clients are currently connected
- **Active API keys** — with last-used timestamp
- **Requests over time** — a 24h / 7d / 30d chart
- **Top tools** — which tools are being called most
- **Recent activity** — latest tool invocations with user, key, status
- **Module health** — green/red for each module
- **License status** — Free / Pro / Agency with renewal date

## Quick actions

- Create API key → `/wp-admin/admin.php?page=storemcp-api-keys&new=1`
- View full activity log
- Test a tool invocation from the browser
- Flush StoreMCP's internal caches
